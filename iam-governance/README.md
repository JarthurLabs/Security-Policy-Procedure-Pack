# IAM Governance

## Purpose

This section defines how Northstar Dental Billing LLC manages identity and access across business applications.

The goal is to reduce:

- Excessive access
- Orphaned accounts
- Privilege creep
- Shared admin usage
- Weak MFA coverage
- Unclear approval ownership

## Core IAM Processes

| Process | Purpose | File |
|---|---|---|
| Joiner / Mover / Leaver Workflow | Manage access throughout employee lifecycle | `joiner-mover-leaver-workflow.md` |
| Access Request Form | Standardize access requests and approvals | `access-request-form.md` |
| Quarterly Access Review | Validate access still matches job need | `quarterly-access-review-process.md` |
| Privileged Account Approval | Control admin and elevated access | `privileged-account-approval-workflow.md` |
| MFA Enforcement Matrix | Define MFA requirements by system | `mfa-enforcement-matrix.md` |
| Role-Based Access Examples | Show practical access by job role | `role-based-access-examples.md` |

## Operating Principle

Access should be granted based on job need, approved by the right owner, reviewed regularly, and removed promptly when no longer required.
