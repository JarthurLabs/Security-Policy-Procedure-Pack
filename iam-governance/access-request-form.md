## Approval Rules

| Access Type | Required Approval |
|---|---|
| Standard role access | Manager |
| Customer data access | Manager + system owner |
| Finance/billing access | Manager + finance owner |
| Admin access | Manager + system owner + operations manager |
| Temporary contractor access | Manager + system owner + expiration date |

## Denial Criteria

Access may be denied when:

- Business justification is unclear
- Access level exceeds job need
- MFA cannot be enforced
- User has not completed required security awareness
- Request lacks system owner approval
