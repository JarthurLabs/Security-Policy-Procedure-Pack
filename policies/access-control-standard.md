# Access Control Standard

## Owner

Operations Manager

## Purpose

Define how access is requested, approved, reviewed, changed, and removed.

## Requirements

- Access must be based on job need.
- Access must be approved before provisioning.
- Privileged access requires enhanced approval.
- Contractors must have expiration dates.
- Departing users must have access removed promptly.
- Access must be reviewed at least quarterly.

## Evidence

Acceptable evidence includes:

- Access request forms
- Admin console exports
- Access review sign-off
- Offboarding checklist
- Exception records
