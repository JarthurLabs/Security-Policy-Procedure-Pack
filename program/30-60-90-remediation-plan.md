# 30 / 60 / 90-Day Remediation Plan

## First 30 Days

Focus: reduce immediate access and authentication risk.

| Action | Owner | Success Criteria |
|---|---|---|
| Enforce MFA for email, admin accounts, password manager, and billing system | IT Administrator | 100% MFA coverage for critical systems |
| Create access request process | Operations Manager | All new access requests use standard form |
| Create leaver process | Operations Manager | Departing users removed same day |
| Identify privileged accounts | IT Administrator | Privileged access register created |

## Days 31-60

Focus: improve review and vendor governance.

| Action | Owner | Success Criteria |
|---|---|---|
| Complete first access review | Operations Manager | Unneeded access removed |
| Launch vendor risk review checklist | Business Owner | New vendors reviewed before approval |
| Build security ownership chart | Operations Manager | Owners assigned for key processes |
| Create policy review calendar | Operations Manager | Review dates assigned |

## Days 61-90

Focus: validate operations and prepare customer assurance evidence.

| Action | Owner | Success Criteria |
|---|---|---|
| Complete technical validation checklist | IT Administrator | Findings documented and assigned |
| Build customer questionnaire response pack | Operations Manager | Responses linked to evidence |
| Run incident response tabletop | Operations Manager | Lessons learned documented |
| Report KPIs to leadership | Operations Manager | Monthly metrics dashboard created |
