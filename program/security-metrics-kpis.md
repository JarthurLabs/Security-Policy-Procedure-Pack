# Security Metrics and KPIs

## Purpose

Security metrics help leadership see whether the program is operating.

| Metric | Target | Owner | Reporting Frequency |
|---|---:|---|---|
| MFA adoption for critical systems | 100% | IT Administrator | Monthly |
| Admin accounts reviewed | 100% quarterly | Operations Manager | Quarterly |
| Departing users removed same day | 100% | IT Administrator | Monthly |
| Open vendor risks | Trending down | Business Owner | Monthly |
| High-risk exceptions past due | 0 | Operations Manager | Monthly |
| Security awareness acknowledgement | 95%+ | Operations Manager | Quarterly |
| Mean incident response triage time | Under 1 business day | Operations Manager | Monthly |
| Access review completion | 100% | Operations Manager | Quarterly |
| Customer questionnaire responses with evidence links | 100% | Operations Manager | Quarterly |

## KPI Notes

Metrics should be simple enough to maintain. A small business should avoid metrics that require more effort to collect than the value they provide.
