## Security Review Questions

| Question | Response | Evidence Needed |
|---|---|---|
| Does the vendor support MFA? |  | Security documentation or admin screenshot |
| Does the vendor support SSO? |  | Security documentation |
| What data will be stored? |  | Data flow summary |
| Does the vendor encrypt data in transit and at rest? |  | Security documentation |
| Does the vendor have incident notification terms? |  | Contract or security page |
| Can data be exported and deleted? |  | Documentation |
| Does the vendor use subprocessors? |  | Subprocessor list |
| Is admin access role-based? |  | Admin documentation |

## Approval Decision

| Decision | Meaning |
|---|---|
| Approved | Vendor can be used |
| Conditional Approval | Vendor can be used only after required controls are confirmed |
| Rejected | Vendor should not be used |
| More Information Needed | Vendor cannot be approved yet |
